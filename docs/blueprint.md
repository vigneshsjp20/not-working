# **App Name**: FlagMaster Quest

## Core Features:

- Dynamic Flag Quiz Generation: Displays country flags, generates 4 randomized multiple-choice country names, ensuring no flag repetition within a game session for over 150 unique flags.
- AI-Powered Educational Hints: A 'Hint' button leverages a generative AI tool to provide fun, educational, and context-rich clues about the displayed country's culture, food, landmarks, or famous facts.
- Real-time Gameplay Feedback: Implements a 10-second timer per question, visually highlights correct and incorrect answers with green/red, reveals the correct answer post-selection, and includes distinct sound effects for responses.
- Score Tracking & Game Restart: Accurately tracks the player's score throughout the game session and offers a clear, easily accessible option to restart and replay the challenge.
- Adaptive UI with Visual Flourishes: Ensures a mobile-friendly and responsive design, featuring engaging animations for correct/wrong answers and smooth transitions between game states, enhancing user interaction.

## Style Guidelines:

- The application employs a light color scheme, reflecting the educational and engaging nature of a world discovery game, now incorporating hints of purple.
- Primary color: A balanced and fresh green (#2EB82E), signifying growth, nature, and the global aspect of the game.
- Secondary color: A soft, light lavender purple (#E0BBE4) providing a calming and educational tone.
- Background color: A very light, desaturated pastel green (#EBFDEB), providing a soft and airy foundation for content while maintaining visual harmony with the primary color.
- Accent color: A vibrant yellow-green (#C7FF1F), used to draw attention to interactive elements, calls to action, and convey energy and excitement.
- Body and headline font: 'Inter', a grotesque-style sans-serif font chosen for its modern, machined, and neutral appearance, ensuring excellent readability across various text sizes.
- Uses clean, flat-style icons for game mechanics and incorporates expressive emojis for fun, instant feedback on answers, reinforcing the game-like and engaging UI.
- Features full-page gradient backgrounds that blend colors harmoniously, providing a modern and immersive backdrop. Interactive elements and information cards utilize a 'glassmorphism' design, creating a contemporary, layered depth effect that ensures content visibility and aesthetic appeal.
- Subtle yet effective animations and transitions are employed for seamless flag changes, question updates, and score tallying. Celebratory animations (e.g., confetti burst) provide positive reinforcement for correct answers, while subtle visual cues (e.g., a slight shake) indicate incorrect responses, making the gameplay engaging and reactive.
